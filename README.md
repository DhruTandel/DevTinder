# DevTinder

A full-stack developer networking platform inspired by Tinder-style matching for developers.

## Features

- JWT Authentication using HTTP-only cookies
- Developer Feed
- Connection Requests (Send / Accept / Reject)
- Real-time Chat using Socket.IO
- Online / Offline Status
- Last Seen Tracking
- Premium Membership Integration using Razorpay
- REST APIs with Express.js
- MongoDB Database
- AWS EC2 Deployment

## Tech Stack

### Frontend
- React.js
- Redux Toolkit
- React Router
- Tailwind CSS

### Backend
- Node.js
- Express.js
- JWT Authentication
- Socket.IO

### Database
- MongoDB
- Mongoose

### Deployment
